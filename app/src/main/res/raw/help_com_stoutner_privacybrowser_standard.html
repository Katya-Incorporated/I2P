Privacy Brower has native I2P support!

Make sure your I2P router is started before launching Privacy Browser

Privacy Browser supports both I2P and Orbot (Tor), but not at the same time.

How to configure Privacy Browser:

  1. Open the main menu and tap “Settings”
  2. Select "Disable Javascript"(Recommended)
  3. Select "Enable Incognito Mode"(Optional)
  4. Tap the "Proxy" menu item and select "I2P" from the submenu
  5. Exit the settings menu.
